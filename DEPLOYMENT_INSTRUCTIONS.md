# Инструкция по развертыванию Google Apps Script

Для корректной работы отправки данных в Google Таблицу, необходимо развернуть Google Apps Script и получить реальный ID скрипта.

## Шаг 1: Создание Google Apps Script

1. Перейдите на сайт [Google Apps Script](https://script.google.com/)
2. Нажмите "Новый проект"
3. Замените содержимое файла Code.gs на содержимое файла Code.gs из этого проекта
4. Нажмите "Сохранить проект" и дайте ему имя, например "Registration Handler"

## Шаг 2: Настройка доступа к Google Таблице

1. Создайте новую Google Таблицу или используйте существующую
2. Скопируйте ID таблицы из URL (это длинная строка символов между `/d/` и `/edit` в URL таблицы)
3. Замените значение переменной `sheetId` в коде на ваш ID таблицы:

```javascript
const sheetId = 'ВАШ_РЕАЛЬНЫЙ_ID_ТАБЛИЦЫ_ЗДЕСЬ';
```

## Шаг 3: Публикация как веб-приложение

1. В редакторе Google Apps Script нажмите "Разработка" → "Новый выпуск"
2. Выберите "Веб-приложение" в поле "Тип"
3. Установите "Кто имеет доступ к скрипту": "Любой, у кого есть ссылка" или "Открытый - доступ для всех"
4. Нажмите "Развернуть"
5. Скопируйте предоставленный URL развертывания

## Шаг 4: Обновление URL в HTML-файле

1. В файле `index.html` замените placeholder URL на реальный ID скрипта из URL развертывания
2. URL будет выглядеть примерно так: `https://script.google.com/macros/s/REAL_SCRIPT_ID/exec`
3. Замените `YOUR_SCRIPT_ID` в строке:
   ```javascript
   const scriptUrl = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```

## Шаг 5: Проверка работы

После замены placeholder на реальный ID скрипта, данные формы регистрации будут отправляться в Google Таблицу.

## Важные замечания

- Убедитесь, что у вас есть права на редактирование Google Таблицы
- При первом запуске скрипт запросит разрешения на доступ к Google Таблицам
- Всегда используйте безопасные соединения (HTTPS)
- Для продакшена рекомендуется использовать более ограниченные права доступа к скрипту
